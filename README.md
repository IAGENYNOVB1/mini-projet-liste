## Fonctionalité disponible

Les fonctionalité disponible pour notre projets ne sont autres que la posibilité d'ajouter des articles.

On a utilisé l'IA pour ajouter la fonctionalité de modification d'article, mais aussi pour mettre en place le CSS afin d'aligner les prochaines modifications qui seront effectuées sur le site du projet.

La possibilité de suprimer des articles à n'importe quel moment en cliquant sur la croix rouge.

Les tests sur le backend se lanceront automatiquement lors de l'initialisation du serveur avec un retour dans le terminal pour chacun d'entre eux.
